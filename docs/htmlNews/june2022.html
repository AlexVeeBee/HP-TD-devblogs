Diagon Alley progress - Ollivander's shop & building

Hello !

This month again I've been working on Ollivander's shop. I'm mostly done with the part that was shown in the movie, so now it's time to go completely off the rails and make everything else that I imagine would be in this building !

It's very interesting as I can be much more imaginative, and it's also a good training for the rest of DIagon Alley, as we ony see a fraction of this place in the movie.
